# Icon

A reusable, accessible Icon atom.

## Usage

```tsx
<Icon icon={GoogleIcon} />
```

> ⚠️ The `icon` prop expects a React component, not a JSX element. So you should pass the imported SVG directly without wrapping it in JSX.
> Don't forget to import your SVGs with the `?react` suffix to ensure they are treated as React components.

## Accessibility

- It uses `aria-hidden="true"` to hide decorative icons from assistive technologies.

- It uses `role="presentation"` for purely decorative icons.

## Example

```tsx
import GoogleIcon from '@Front/assets/svg/google_icon.svg?react';
import PaletteIcon from '@material-symbols/svg-400/outlined/palette.svg?react';

<Icon icon={GoogleIcon} />
<Icon icon={PaletteIcon} className="custom-color" />
```
