# Spinner

A reusable, accessible Spinner atom.

## Usage

```tsx
<Spinner />
```

## Accessibility

- It uses `aria-hidden="true"` to hide decorative spinner from assistive technologies.

- It uses `role="presentation"` for purely decorative spinner.

- It uses `aria-live="polite"` to announce state change without interupting user.

- It uses `aria-busy="true"` to indicate that an area is loading.
