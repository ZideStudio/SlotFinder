# Link

A reusable, accessible link atom for navigation and external resources.

## Usage

```tsx
<Link href="https://react.dev">
  React documentation
</Link>

<Link href="https://react.dev" openInNewTab>
  React documentation (new tab)
</Link>
```

## Accessibilty

- Follows WCAG 2.2 AA and RGAA 4 guidelines.

- Fully keyboard accessible.

- Visible focus state when navigating with keyboard.

- Uses secure defaults when opening links in a new tab.

## Security

- Automatically applies <code>rel="noopener noreferrer"</code> when <code>openInNewTab</code> is enabled.

- Prevents tab-nabbing vulnerabilities.

## Example

```tsx
<Link href="/home">
  Internal navigation
</Link>

<Link href="https://external-site.com" openInNewTab>
  External website
</Link>

```
