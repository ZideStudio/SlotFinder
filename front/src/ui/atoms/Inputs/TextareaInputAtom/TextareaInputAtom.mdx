# TextareaInputAtom

A reusable, accessible textarea input atom for forms and UI elements.

## Usage

```tsx
<TextareaInputAtom name="input-name" placeholder="Enter text" className="custom-class" />
```

## Accessibility

- Follows WCAG 2.2 AA and RGAA 4 guidelines.
- Keyboard accessible.
- Use with a `<label htmlFor>` for best accessibility.

## Example

```tsx
<label htmlFor="my-input">Label</label>
<TextareaInputAtom name="my-input" id="my-input" placeholder="Type here..." />
```
