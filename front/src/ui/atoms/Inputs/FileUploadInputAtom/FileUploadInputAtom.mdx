# FileUploadInputAtom

A reusable file upload input atom that allows users to select files from their device.

---

## Usage

```tsx
<FileUploadInputAtom name="file-upload" className="custom-class" />
```

# Behavior

- Allows users to select files from their device.

- Clicking the component opens the native file picker.

---

# Accessibility

- Uses a native `<input type="file">` for optimal browser and assistive technology support.

- The input is associated with a `<label>` via a generated id.

- Decorative icon is hidden from screen readers (aria-hidden="true").

---
