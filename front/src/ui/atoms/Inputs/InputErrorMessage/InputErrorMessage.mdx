# InputErrorMessage

A reusable, accessible error message for form inputs and UI elements.

## Usage

```tsx
<InputErrorMessage id="error-input" message="Error message" />
```

## Accessibility

- Follows WCAG 2.2 AA and RGAA 4 guidelines.
- Required fields are visually indicated and use <code>aria-hidden</code> for the asterisk.

## Example

```tsx
<div>
  <label htmlFor="email-input">Email address</label>
  <input id="email-input" type="email" required aria-describedby="email-error" />
  <InputErrorMessage id="email-error" message="Please enter a valid email address" />
</div>
```
