# CheckboxInputAtom

A reusable, accessible checkbox atom for forms and UI elements.

## Usage

```tsx
<CheckboxInputAtom name="example" />
```

## Accessibility

- Follows WCAG 2.2 AA and RGAA 4 guidelines.
- Keyboard accessible.

## Example

```tsx
<CheckboxInputAtom id="my-input" name="terms" required />
<label htmlFor="my-input">Label</label>
```
