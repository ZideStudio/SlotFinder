# CheckboxInputAtom

A reusable, accessible checkbox atom for forms and UI elements.

## Usage

```tsx
<CheckboxInputAtom id="example" name="example" />
```

## Accessibility

- Follows WCAG 2.2 AA and RGAA 4 guidelines.
- Keyboard accessible.
