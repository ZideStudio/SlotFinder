# SelectInputAtom

A reusable, accessible select input atom.

## Usage

```tsx
<SelectInputAtom
  id="my-select"
  name="my-select"
  placeholder="Choose one"
  options={[
    { label: 'Option A', value: 'A' },
    { label: 'Option B', value: 'B' },
  ]}
/>
```
