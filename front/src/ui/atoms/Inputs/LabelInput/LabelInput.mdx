# LabelInput

A reusable, accessible label atom for form inputs and UI elements.

## Usage

```tsx
<LabelInput inputId="username">Username</LabelInput>
```

## Accessibility

- Follows WCAG 2.2 AA and RGAA 4 guidelines.
- Keyboard accessible.
- Label is programmatically associated with input via <code>htmlFor</code> and <code>id</code>.
- Required fields are visually indicated and use <code>aria-hidden</code> for the asterisk.

## Example

```tsx
<LabelInput inputId="email-input" required>
  Email address
</LabelInput>
<input id="email-input" type="email" />
```
