# OverlayContent

A reusable content shell shared by overlay components such as `Modal` and `Popover`. It renders a structured layout with a header (title + close button), a main body, and a footer (primary and optional secondary actions).

## Usage

```tsx
import { OverlayContent } from '@Front/ui/organisms/OverlayContent/OverlayContent';

const Example = () => (
  <OverlayContent
    title="Confirmation"
    primaryButtonProps={{
      children: 'Confirm',
      onClick: () => {},
    }}
    secondaryButtonProps={{
      children: 'Cancel',
      variant: 'secondary',
      onClick: () => {},
    }}
    closeOverlay={() => {}}
  >
    Are you sure you want to proceed?
  </OverlayContent>
);
```

## Accessibility

- The title is rendered as an `<h1>` and can be linked to the parent overlay via `aria-labelledby` using `titleId`.
- The close button has an explicit accessible name (`Fermer la fenêtre`).
- Uses semantic structure: `<header>` for the title area, `<div>` for the body, and `<footer>` for actions.
- `<header>` and `<footer>` lose their global `banner`/`contentinfo` landmark roles when nested inside a sectioning root such as `<dialog>`. When `OverlayContent` is used inside a non-sectioning element (e.g., `<div popover>` for a future Popover), prefer replacing them with `<div>` elements to avoid unintended global landmarks.
- All interactive elements are keyboard-operable through native button semantics.

## Portal rendering

Overlay components that use `OverlayContent` (such as `Modal`) should be rendered via a **React Portal** into `document.body`. Refer to each component's own documentation for implementation details.

## Layout notes

`OverlayContent` does not manage the gap between the header, body, and footer. This is intentional: in the case of a `<dialog>`, spacing should only be applied when the element is open (e.g. via `&[open] { gap: ... }`). The parent component is responsible for defining this spacing.

```scss
// Example in Modal.scss
.ds-modal {
  &[open] {
    display: flex;
    flex-direction: column;
    row-gap: 0.75rem;
  }
}
```
