# ClickIcon

A reusable ClickIcon molecule for Ui elements. The ClickIcon component supports polymorphism through the `as` prop, allowing it to render as different HTML elements while maintaining consistent styling and behavior.

## Usage

```tsx
<ClickIcon icon={GoogleIcon} />
```

### As a Link

```tsx
<ClickIcon as="a" icon={SendIcon} href="/about" />
```

### As a Span (for custom interactions)

```tsx
<ClickIcon
  as="span"
  icon={SendIcon}
  onClick={() => console.log("Span clicked")}
/>
```

## Example

```tsx
import SendIcon from '@material-symbols/svg-400/outlined/send.svg?react';
import GoogleIcon from '@Front/assets/svg/google_icon.svg?react';

<ClickIcon icon={SendIcon} />
<ClickIcon icon={GoogleIcon} className="custom className" />
<ClickIcon as="a" icon={SendIcon} href="/about" />
```
