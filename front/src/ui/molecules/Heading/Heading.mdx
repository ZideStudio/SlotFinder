# Heading

A reusable heading component for consistent page and section titles.

## Usage

```tsx
<Heading level={1}>Main page title</Heading>
```

## Accessibility

- Use heading levels in order when possible.
- Prefer a single `h1` per page.
- Keep headings clear and descriptive.
