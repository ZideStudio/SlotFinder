# Button

A reusable, accessible Button component for interaction. The Button component supports polymorphism through the `as` prop, allowing it to render as different HTML elements while maintaining consistent styling and behavior.

## Usage

```tsx
<Button>
  Button content
</Button>

<Button className="custom-class" icon={GoogleIcon}>
  Button content with custom class
</Button>
```

### As a Link

```tsx
<Button as="a" href="/about">
  Go to About
</Button>
```

### As a Link with external URL

```tsx
<Button as="a" href="https://example.com" target="_blank" rel="noopener noreferrer">
  Visit Example
</Button>
```

### As a Span (for custom interactions)

```tsx
<Button as="span" onClick={() => console.log('Span clicked')}>
  Custom span button
</Button>
```
