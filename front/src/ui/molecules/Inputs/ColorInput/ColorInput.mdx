# ColorInput

A reusable, accessible color input for forms and UI elements.  
Combines the `ColorInputAtom` with a label and error message via the `Field` utility component.

## Usage

```tsx
<ColorInput label="Input Label" name="color-input" description="Choisir une couleur" className="custom-class" />
```

## Accessibility

- Follows WCAG 2.2 AA and RGAA 4 guidelines.
- Keyboard accessible.
- Error message is linked to the input via `aria-describedby`.
