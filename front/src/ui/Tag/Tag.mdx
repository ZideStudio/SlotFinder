# Tag

A reusable, accessible tag atom for Ui elements.

## Usage

```tsx
<Tag>Tag text</Tag>
```

## Accessibility

- The title display the children text.

## Example

```tsx
<Tag color="#007bff">Tag text</Tag>
```

## Long text behavior

When the content is too long, the tag will be truncated with an ellipsis and the full text will be available on hover via the `title` attribute.
