# Field

A reusable, accessible text input atom for forms and UI elements.

## Usage 

```tsx
<Field input={TextInputAtom} className="custom-class" />
```

- Follows WCAG 2.2 AA and RGAA 4 guidelines.
- Keyboard accessible.
